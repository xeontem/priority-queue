### Priority queue task

### Prerequisites
* Install [nodejs](https://nodejs.org/en/) (>= v6.2.0)
* open bash in this folder
* `npm install`

### Run tests
```sh
npm test
```
### Run tests in watch mode
```sh
npm test:watch
```
### Run only specified tests
- just add ".only" to describe you want to test, e.g.:
````
describe.only('Node', () => {...
````

### Run in browser
```sh
npm start
```

open http://localhost:8080
